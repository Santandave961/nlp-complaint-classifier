# nlp-complaint-classifier

## Project Overview
A Natural Language Processing (NLP) project that automatically classifies financial customer complaints into product categories using Machine Learning. This eliminates manual complaint routing and improves response times for customers.

### Business Problem
Financial Institutions recieve thousands of customer complaints daily. Manually reading and routing each complaint to the right department is:
- Time consuming
- Prone to human error
- Costly at scale.
This model automates that process with 87% accuracy

### Dataset
Source: Consumer Financial Protection Bureau(CFPB)
Size: 160000+ complaints
Features: Complaint narrative text, Product Category
Categories:
- Credit reporting
- Credit Card
- Debt Collection
- Mortgages and Loans
- Retail Banking

# Business Impacts
- Auto routes complaints to correct department instantly.
- Reduces manual sorting workload significantly.
- 87% accuracy means fewer misrouted complaints.
- Scales to handle thousands of complaints per day.
